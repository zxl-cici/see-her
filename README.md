# See HER — 遗物整理师

> 你是遗物整理师：翻阅档案、拼凑遗物、还原真相，最后与那位被历史写小的女性面对面。

## 游戏简介

一款手机网页互动游戏，扫码即玩，无需下载。

画风像一本旧时代童话科学档案——手绘彩铅质感，旧纸纹理，细黑线勾边，安静、温柔，带一点神秘。

主角：**Rosalind Franklin（罗莎琳德·富兰克林）**——用 X 射线晶体学拍下"Photo 51"的科学家，DNA 双螺旋结构发现的关键贡献者，却在诺贝尔奖致辞中只字未提。

## 游戏流程

| 阶段 | 内容 | 操作 |
|------|------|------|
| 开场 | 观看开场视频 | 观看，结束后点击按钮进入 |
| 第一关 | 碎片拼图 — 拼回被撕碎的讣告 | 拖拽碎片到正确位置 |
| 第二关 | 底片对焦 — 让 Photo 51 从模糊变清晰 | 滑动对焦环 |
| 第三关 | 漫画还原 — 还原被打乱的实验场景 | 拖拽漫画格到正确顺序 |
| 终章 | 与她面对面对话 | 点击选项进行对话 |

## 本地运行

```bash
# 直接打开 index.html 即可
start index.html

# 或使用任意 HTTP 服务器（推荐，音效加载更稳定）
npx serve .
```

## 文件结构

```
see-her/
├── index.html          # 主入口 — 完整互动游戏（~27MB，base64 内嵌所有资源）
├── ending.html         # 终章对话独立版
├── README.md
├── 游戏介绍文案.md       # 完整游戏叙事文案
├── 游戏操作指南.md       # 操作说明
├── figure-database.md  # 52位非典型女性先驱档案数据库
└── assets/
    ├── act1/           # 第一幕：碎片拼图素材
    ├── act2/           # 第二幕：底片对焦素材
    ├── act3/           # 第三幕：漫画还原素材
    ├── ending/         # 终章：人物立绘（首/尾）
    ├── video/          # 开场 & 结尾视频
    └── screenshots/    # 项目展示截图
```

## 技术

- 纯静态 HTML/CSS/JavaScript
- 移动端优先，适配 375-430px 宽度
- 拖拽、滑动对焦、漫画排序三种交互模式
- 打字机效果对话框
- 所有视觉资源 base64 内嵌
